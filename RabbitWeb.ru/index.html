Дизайн

Типовой или Индивидуальный?

Типовой дизайн сайта – это технология создания сайта на основе готового универсального шаблона многократного использования.
Типовой дизайн сайта – это бюджетный вариант сайта.
Стоимость типового дизайна гораздо ниже индивидуального.
Все пакетные предложения изначально предполагают типовой дизайн.
Это значит, что при получении заказа на создание сайта (заполненной бриф-формы) мы подбираем шаблон для сайта из готовых универсальных шаблонов.

Подробнее..

Индивидуальный дизайн сайта – это технология создания сайта на основе уникального шаблона, специально разрабатываемого только для одного, конкретно взятого web-проекта.
При создании индивидуального дизайна сайта никакие готовые шаблоны не используются.
Как правило, такой шаблон создается "с нуля".
