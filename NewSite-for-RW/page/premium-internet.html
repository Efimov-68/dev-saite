CЕМАНТИЧЕСКОЕ ЯДРО

Это перечень поисковых слов, их словосочетаний и морфологических форм, наиболее точно характеризующих товары или услуги, которые предлагает интернет-магазин, и специально подобранных для его продвижения.

Семантическое ядро используется специалистами по оптимизации для проработки максимально эффективной структуры каталога. Вложив нужным образом ВЧ\СЧ\НЧ-запросы друг в друга, вы получите каталог, который будет "ловить огромной сетью" посетителей поисковых систем :)

  • - Подбор и группировка рентабельных ключевых запросов (семантического ядра) для продвижения в поисковых системах;
  • - Разработка структуры каталога товаров, оптимизированной для дальнейшего продвижения в поисковых системах (часть запросов линкуется с подразделами каталога товаров, часть запросов — с соответствующими фильтрами в этих подразделов).

Результат: Утвержденное семантическое ядро (СЯ), утвержденная структура веб-сайта и сопроводительная записка в формате.docx, которые являются основой для проектирования веб-сайта.

Примечание: данные документы нужны для понимания, под какие запросы делается интернет-магазин, какие подразделы каталога нужно специально выделить в структуре сайта, какие фильтры необходимы предусмотреть для соответствия средне- и низко-частотным запросам в поисковых системах, каким будет контент и сколько его необходимо разместить на страницах приземления.

При таком подходе на выходе производства интернет-магазин мы получаем готовый к полноценному поисковому продвижению с наращиванием ссылочной массы и индекса доверия.

ЗАЧЕМ НУЖЕН ЧЁРНО-БЕЛЫЙ САЙТ?

Невозможно разработать качественный интернет-магазин без решения целого ряда вопросов о контенте (наполнении каталога товаров).

Поэтому мы настоятельно рекомендуем сначала запрограммировать и наполнить чёрно-белый сайт (и/или интегрировать черно-белый сайт с 1С), а потом заниматься вопросами оформления (выбирать шрифты/цвета/имиджевые картинки и проч).

ЗАЧЕМ НУЖЕН ДИЗАЙН ВНУТРЕННИХ СТРАНИЦ?
