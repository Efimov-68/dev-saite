Для детей

Scratch

Scratch - это среда программирования позволяющая детям создавать собственные интерактивные проекты: игры, мультфильмы, обучающие системы, комиксы и т.д. Язык программирования, который создавался специально для детей и подростков (8 – 16 лет). Позволяющий технологически выразить себя в компьютерном творчестве. Одно из преимуществ языка – это интуитивная понятность и простота в использовании.

Python

Python — простой в изучении язык программирования, он особенно хорош для начинающих. В отличие от многих других языков, код написанный на языке Python легко читается, а интерактивная оболочка позволяет вводить программы и сразу же получать результат. Помимо простой структуры языка и интерактивной оболочки, в Python есть инструменты, заметно ускоряющие обучение и позволяющие создавать несложные анимации для своих видеоигр.

JavaScript

JavaScript — это один из популярных языков программирования, который используется повсюду, его поддерживают браузеры Chrome, Firefox и Internet Explorer. А освоив язык программирования, Ваш ребенок или Вы станете программистом — человеком, который не просто пользуется компьютерами, а управляет ими. Научившись программированию, вы сможете вертеть компьютерами как хотите, и они всегда будут послушно следовать вашим указаниям.
